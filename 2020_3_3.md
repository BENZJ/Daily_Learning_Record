# 今日收获



## top-1和top-5的概念
对于Imagenet数据分类总是出现top-1错误率和top-5错误率
- top-1是模型只给出一个预测结果
- top-5是模型给出5个模型预测结果。

这两个都是评判模型好坏的重要指标。
